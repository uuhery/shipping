#  🚚物流运输系统
## 🥇技术栈
- 后端：
- 微服务架构：Spring Cloud Alibaba + Nacos + Gateway + Hystrix
- 数据库：Druid + Spring AOP
- 定时任务管理：Quartz
- 身份验证：OAuth 2.0
- 数据校验：Hibernate Validator
- 前端：Thymeleaf + Layui
- 实训小作业集成vue前端 项目链接：https://gitee.com/uuhery/logistics-management-system
